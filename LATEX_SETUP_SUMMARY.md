## ğŸ› TROUBLESHOOTING

### Error: "pdflatex not found"
1. Reinstall MiKTeX from https://miktex.org
2. Restart computer
3. Try again

### Error: "File not found"
1. Check you're in correct directory
2. Verify `JobSphere_Presentation.tex` exists in that folder
3. Check filename spelling

### Slides look weird
1. Close PDF and recompile
2. Make sure you ran pdflatex TWICE
3. Check for typos in LaTeX code (usually missing `}`)

### PDF won't open
1. Check compilation log for errors
2. Delete all temporary files (`.log`, `.aux`, `.out`)
3. Recompile fresh

### Colors look wrong
1. Save and close PDF
2. Recompile
3. Note: Projectors display colors differently than monitors
